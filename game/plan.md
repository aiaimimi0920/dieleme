# 游戏策划书：《房价真相 / The Price of Truth》

> "当泡沫破裂时，唯一剩下的就是重力。"

## 1. 核心概念 (Core Concept)
- **定位**: 一个带有黑色幽默色彩的数据可视化工具/“反向”炫富小程序。
- **核心比喻**: "牛马" (The Wage Slave) —— 努力工作却在这个负和游戏中挣扎。

## 2. 游戏流程 (Game Flow)

### 2.1 第一关：切蜡烛 (Candle Cutting) - 过去/比例
*   **交互**: 巨大的红蜡烛，用户滑动切掉“跌幅”。
*   **隐喻**: “燃烧殆尽的资产”。
*   **反馈**: “太悲观 / 没数 / 人间清醒”。

### 2.2 第二关：虚无天平 (The Scale of Void) - 现在/精准
*   **交互**: 左侧未知亏损 vs 右侧实物（特斯拉、喜茶、青春）。
*   **反馈**: 天平平衡瞬间，亏损数字显现，生成痛苦清单。

### 2.3 第三关：命运轮盘 (The Wheel of Destiny) - 未来/玄学 🎡
既然我们无法预测未来，那就交给命运。

*   **核心逻辑**：**在荒谬的现实中寻找一点虚假的安慰 (Comforting Lies vs Brutal Truth)**
*   **场景**:
    *   一个巨大的、花花绿绿的**博彩轮盘**。
    *   **奖池分布**:
        *   **90% 区域 (吉祥话)**: "时来运转", "财源滚滚", "利空出尽", "触底反弹", "家和万事兴", "稳字当头"。
        *   **10% 区域 (真实预言)**: **"明年再跌 20%"**, **"腰斩在即"**, **"还得跌"** (基于真实数据预测)。
*   **交互**:
    *   用户用力拨动轮盘。
    *   轮盘飞速旋转，发出咔哒咔哒的声音。
    *   最后缓缓停下。
*   **反馈**:
    *   **抽中吉祥话**: 满屏烟花，系统恭喜您获得了“精神胜利法”大奖。
    *   **抽中真实预言**: 只有一声悲凉的鸦叫。系统提示：“很遗憾，您抽中了唯一的真相。”

## 3. 终极产物：牛马海报 (The Niu Ma Poster) 🖼️
### 3.1 视觉布局
| 区域 | 内容 | 备注 |
| :--- | :--- | :--- |
| **顶部** | **轮盘结果 (Title)** | **抽中的那句吉祥话 (或诅咒)**<br>e.g. "财源滚滚" (具有强烈的反讽效果) |
| **中心** | **牛马状态** | **吉祥物 + 职业装**<br>- *微亏*: 开滴滴<br>- *中亏*: 送外卖<br>- *巨亏*: 敲代码 |
| **左上** | **🕯️ 蜡烛** | 认知偏差状态 |
| **右上** | **⚖️ 亏损实物** | 现在的痛苦清单 |
| **左下** | **📉 明年预警** | **"系统预测: 跌幅 15%"**<br>(无论轮盘抽中什么，这里都会悄悄显示真实预测，作为彩蛋/补充) |
| **右下** | **📱 二维码** | "转动命运的齿轮，测测你的运气？" |

## 4. 技术与设计 (Tech & Design)
- **前端**: Vue3/React + Matter.js + Canvas (轮盘绘制)。
- **数据**:
    - `config.json` 增加 `Wheel_Phrases` (吉祥话列表)。
    - `Prediction_Algorithm`: 简单的趋势外推算法。
